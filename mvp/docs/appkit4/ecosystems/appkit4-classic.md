# AppKit 4 (Classic)

Default PwC design system theme — blue primary actions, standard PwC branding.

## CSS / themes

| Source | Use |
|--------|-----|
| **npm** `@appkit4/styles` | SCSS themes (recommended) — see [styles/styles.md](../styles/styles.md) |
| **CDN** | `appkit.min.css`, dark and accent variants — [portal CDN links](../references/portal-links.md#themes-and-fonts) |

## Angular

- Import Classic theme in `angular.json` per [styles/README.md](../styles/README.md).  
- Components from `@appkit4/angular-components` — no ecosystem flag in Angular package; theme is CSS-driven.

## Preference center (portal)

Classic is the first option in the AppKit preference center (blue selection state in the live portal).

## When to use

- Standard PwC internal and client-facing apps unless Re-Branded is mandated.
