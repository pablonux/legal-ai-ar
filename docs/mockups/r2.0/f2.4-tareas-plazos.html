¿Qué hace?

Tablero kanban de tareas del proyecto con asignación, prioridad y fechas. Franja de plazos regulatorios/contractuales con cómputo de días hábiles y alertas automáticas al acercarse el vencimiento.

Flujo

  1. Manager crea tareas y las asigna con fecha y prioridad
  2. Equipo mueve las tarjetas entre columnas según avance
  3. Sistema alerta cuando una tarea se acerca al vencimiento
  4. Plazos regulatorios se muestran en la franja inferior
  5. Manager revisa el estado en un vistazo desde el kanban
