¿Qué hace?

Panel de trazabilidad de llamadas a modelos de IA: latencia P50/P95, tokens consumidos, versiones de modelo, costo proyectado y alertas. Trazas detalladas por usuario y agente para debugging y optimización.

Flujo

  2. Ve KPIs: latencia, tokens, costo
  3. Revisa las trazas recientes para identificar problemas
  4. Recibe alertas automáticas si se superan umbrales
  5. Drill-down a una traza específica para debugging
