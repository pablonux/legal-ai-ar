¿Qué hace?

Alta/baja de usuarios y asignación de roles globales. Gestión de fuentes de datos activas (monitoreo de ingesta). Configuración del sistema. Log de auditoría inmutable de todos los cambios de configuración.

Flujo

  1. Admin accede al panel con tabs: Usuarios / Fuentes / Config / Auditoría
  2. Gestiona usuarios: rol, proyectos asignados, estado
  3. Monitorea el estado de cada fuente de ingesta
  4. Accede al log de auditoría para revisiones de compliance
