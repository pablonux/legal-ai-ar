¿Qué hace?

Login automático con la cuenta PwC corporativa (Entra ID). Sin usuario ni contraseña manual. GCaaS emite una cookie de sesión HTTP-only segura que la API valida en cada request.

Flujo

  1. Usuario accede a la URL de la app
  2. Plataforma detecta sesión Entra activa y emite cookie segura
  3. API valida la cookie y obtiene perfil + rol del usuario
  4. App muestra el Home personalizado según el rol
  5. Al cerrar sesión, la plataforma invalida la cookie
