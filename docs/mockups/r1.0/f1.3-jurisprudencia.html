¿Qué hace?

Búsqueda híbrida (BM25 + vectorial) sobre jurisprudencia nacional: CSJN, Tribunal Fiscal, Cámaras. Filtros por tribunal, materia, período y resultado. Resultados rankeados con resumen generado por IA y citas destacadas.

Flujo

  1. Usuario escribe consulta en lenguaje natural
  2. Sistema ejecuta búsqueda híbrida BM25 + vectorial
  3. Resultados rankeados con resumen y fuente
  4. Usuario refina con filtros (tribunal, período, materia)
  5. Selecciona un fallo para ver el texto completo o lo guarda en el proyecto
