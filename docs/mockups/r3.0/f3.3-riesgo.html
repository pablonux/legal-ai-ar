¿Qué hace?

Evaluación de riesgo legal/tributario/compliance de un escenario planteado. Puntaje visual por dimensión (bajo/medio/alto) con análisis fundamentado en la KB y recomendaciones prioritarias. Versionado histórico.

Flujo

  1. Usuario describe el escenario y selecciona las áreas a evaluar
  2. IA analiza riesgo legal, tributario y de compliance
  3. Muestra puntaje por dimensión con fundamentación
  4. Lista recomendaciones priorizadas (rojo/amarillo/verde)
  5. El análisis se guarda con fecha para comparar versiones
