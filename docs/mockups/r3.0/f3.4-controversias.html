¿Qué hace?

Seguimiento de controversias tributarias ante ARCA/Tribunal Fiscal/ARBA: organismo, etapa, monto en disputa, plazos computados y alertas automáticas. Documentos y jurisprudencia vinculados al expediente.

Flujo

  1. Usuario registra la controversia (organismo, impuesto, monto)
  2. Define la etapa procesal y el sistema computa plazos
  3. Sistema alerta antes de cada vencimiento procesal
  4. Usuario vincula documentos y jurisprudencia relevante
  5. Manager ve el panel consolidado de todas las controversias
