# Versions and compatibility

## Target for this documentation

| Item | Version |
|------|---------|
| Angular | **19.x** |
| `@angular/cdk` | **19.x** (match core) |
| `@appkit4/angular-components` | **4.31.x** (latest compatible with the project) |
| `@appkit4/styles` | 4.11.x |

## AppKit ↔ Angular matrix (summary)

| AppKit | Angular |
|--------|---------|
| **4.31.0** (recommended) | 19, 20 |
| 4.22.3 | 17, 18 |

For Angular 17–18 projects, pin AppKit **4.22.3** instead of 4.31.x.

## Ecosystems

| Ecosystem | Use when |
|-----------|----------|
| **AppKit 4** (Classic) | Default PwC products — blue primary palette |
| **AppKit 4 Re-Branded** | Products using the re-branded theme (orange accent in preference center) |
| AppKit New Era | CT&I only — **out of scope** unless explicitly requested |

See [ecosystems/README.md](../ecosystems/README.md).
