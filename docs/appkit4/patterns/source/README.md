# React pattern reference (extracted)

TypeScript/SCSS examples extracted from portal **Download source code** ZIPs. **Adapt to Angular** — do not copy JSX into production templates.

## Refresh from portal

1. Open [pattern list](https://appkit.pwc.com/appkit4/content/pattern-list)  
2. Per pattern → **Download source code**  
3. Extract with [../../scripts/extract-pattern-zips.py](../../scripts/extract-pattern-zips.py) (local `import-zips/` folder)

Figma files are **not** stored in the repo — use **Get Figma file** on each pattern detail page.
