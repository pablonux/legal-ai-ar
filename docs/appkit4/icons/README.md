# Icons

AppKit uses a single icon font / class system. Icon names must be validated before use.

## Offline catalog

Alphabetical search exports (MCP):

`icons/icons-a.md` … `icons-z.md`

Search by letter file when looking up `icon-*` class names.

## Usage in Angular

- Apply documented icon classes on `<span>` or AppKit components that accept `icon` inputs.  
- Pair with accessible labels (`aria-label`) when the icon is interactive.

## Portal

https://appkit.pwc.com/appkit4 — icon search in the live design system (requires login).

## AI rule

Never assume an icon exists — confirm in the offline letter file or portal first.
