# Cursor Agent Skills — Legal AI AR

> **Canonical location**: [`.cursor/skills/`](../../../.cursor/skills/)

All agent skills (Analyst, Architect, Designer, Developer, Documenter, Manager, Reviewer) are maintained as Cursor project skills. The prompt copies that previously lived in this folder have been removed — use the skills directly.

## How to Invoke

Start a chat and describe what you need. Cursor discovers skills by description:

- "Do the next development task from the roadmap"
- "Create a work item for [feature idea]"
- "Create mockups for [feature or page]"
- "Review the next unaudited deliverable"
- "Do the next documentation task"

## Key Paths

- **Skills**: `.cursor/skills/{agent}/SKILL.md`
- **Roadmap**: `docs/roadmap/ROADMAP.md`
- **Work Items**: `docs/roadmap/work-items/`
- **Design**: `docs/design/`
- **Mockups**: `docs/mockups/`
