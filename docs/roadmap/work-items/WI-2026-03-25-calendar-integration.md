# Work Item — Calendar Integration

**Created**: 2026-03-25
**Status**: Draft
**Feature**: Integrations

## User Story

As a **lawyer**, I want **to link important dates and deadlines from my cases to my calendar (Outlook, Google Calendar)** so that **I never miss a procedural deadline and can plan my workload around upcoming case milestones**.

## Context

- **Current state**: No calendar or deadline tracking exists in the platform.
- **Target state**: Users can:
  - Set reminders on rulings (e.g. "Review appeal deadline" with a date).
  - Set reminders on search alerts (e.g. "Check for new rulings on Topic X every Monday").
  - Export reminders to external calendars via iCal (.ics) file download or direct Outlook/Google Calendar API integration.
  - View upcoming deadlines in a compact widget on the welcome page.
- **Prerequisites**: Notification system (Phase 3 roadmap) for in-app reminders.
- **Reference**: Syllo deadline management, Filevine calendar integration, practice management tools.
