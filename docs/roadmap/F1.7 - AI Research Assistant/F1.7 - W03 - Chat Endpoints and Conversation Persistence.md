# F1.7 - W03 - Chat Endpoints and Conversation Persistence

> **Feature:** F1.7 - AI Research Assistant
> **Release:** 1.0 | **Sprint:** S01
> **Type:** backend | **Priority:** High
> **Estimate:** 5 story points
> **Assignable to:** Backend Dev

---

## Description

Align the MVP chat endpoint to Minimal API + Contracts, enable conversation persistence with
platform-auth user IDs, and expose conversation management endpoints. The SSE streaming logic
from the MVP is largely reused — this WI focuses on alignment and persistence.

---
