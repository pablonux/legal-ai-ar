## Description

Extend the MVP's generic legal assistant with:
1. A **tax-tuned system prompt** (`tax-legal-assistant.yaml`) in Spanish.
2. A new **`SearchTaxSourcesPlugin`** for SK that queries `idx-tax-sources` (F1.5).
3. Refactored plugin classes aligned to Clean Architecture (`LegalAiAr.Agents` project).
4. An agent eval baseline in `LegalAiAr.AgentEvals`.

### SK Plugins

- [ ] **Create** `backend/src/shared/LegalAiAr.Agents/Plugins/SearchTaxSourcesPlugin.cs`
    ```csharp
    public sealed class SearchTaxSourcesPlugin(ITaxSourcesSearchService searchService)
    {
        [KernelFunction, Description("Busca dictámenes, fallos del TFN, resoluciones de Comisión Arbitral e informes de ARBA.")]
        public async Task<string> SearchTaxDocs(
            [Description("Consulta en lenguaje natural")] string query,
            [Description("Organismo: ARCA, ARBA, TFN, ComisionArbitral (opcional)")] string? organism = null,
            [Description("Rama tributaria: IVA, Ganancias, ISIB, Sellos (opcional)")] string? taxBranch = null)
        {
            var results = await searchService.SearchAsync(new SearchTaxSourcesQuery(query, organism, taxBranch, PageSize: 5));
            return SerializeForContext(results);  // compact JSON for context
        }

        [KernelFunction, Description("Obtiene el texto completo de un documento tributario por ID.")]
        public async Task<string> GetTaxDocDetail(
            [Description("ID del documento tributario")] int taxDocumentId) { ... }
    }
    ```
