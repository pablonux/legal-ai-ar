# F1.7 - W04 - Frontend Chat Feature

> **Feature:** F1.7 - AI Research Assistant
> **Release:** 1.0 | **Sprint:** S01
> **Type:** frontend | **Priority:** High
> **Estimate:** 8 story points
> **Assignable to:** Frontend Dev (senior)

---

## Description

Build the Angular 19 AI research assistant chat UI at `/asistente`. Conversation sidebar, streaming
message rendering, inline citations with navigation links, and context seeding from norm/ruling
detail pages. This is the most complex frontend feature in R1.0.

---

## Tasks

### Feature scaffold

- [ ] Generate `frontend/projects/features/assistant/` via `schema-la:feature`

### Models

- [ ] **Create** `assistant.model.ts`
    ```typescript
    export type MessageRole = 'user' | 'assistant';
    export interface ChatMessage {
      id?: string; role: MessageRole; text: string;
      citations?: CitedSource[]; isStreaming?: boolean;
    }
    export interface CitedSource { type: string; id: string; reference: string; date?: string; url?: string; }
    export interface ConversationSummary { id: string; title: string; lastMessageAt: string; }
    ```

### Services

- [ ] **Create** `assistant.service.ts`
    - `readonly messages = signal<ChatMessage[]>([])`
    - `readonly isStreaming = signal(false)`
    - `sendMessage(text: string, contextEntityType?: string, contextEntityId?: string): void`
        - Opens `EventSource` to `POST /api/chat` (via fetch + ReadableStream — SSE with POST body)
        - On `token` event: append text to last message's `text` signal
        - On `citations` event: set `citations` on last message
        - On `done` event: set `isStreaming(false)`; close stream
    - `startNewConversation(): void`
    - `loadHistory(conversationId: string): void`

    > **Note:** Native `EventSource` does not support `POST`. Use `fetch` + `response.body.getReader()` to
    > stream SSE from a POST endpoint. Parse `data:` lines manually.

- [ ] **Create** `conversation.service.ts`
    - `readonly conversations = signal<ConversationSummary[]>([])`
    - `load(): void`, `delete(id: string): void`

### Components

- [ ] **Create** `assistant-shell.component.ts`
    - Two-panel layout: conversation sidebar (left, collapsible) + chat window (right)
    - "Nueva conversación" button at top of sidebar

- [ ] **Create** `conversation-sidebar.component.ts`
    - List of `ConversationSummary`; click → `assistant.service.loadHistory(id)`
    - Delete icon per conversation (confirm dialog)
    - Empty state: "No hay conversaciones"

- [ ] **Create** `message-list.component.ts`
    - Scrollable container; auto-scroll to bottom when new tokens arrive
    - Renders alternating `message-bubble` components (user right; assistant left)
    - When `isStreaming=true`, shows streaming indicator (animated dots after last token)

- [ ] **Create** `message-bubble.component.ts`
    - User bubble: plain text in AppKit 4 chip/card style; right-aligned
    - Assistant bubble: markdown rendering (use `marked` or simple line-break rendering);
      citation chips below the text
    - Streaming state: cursor blink after last character

- [ ] **Create** `citation-chip.component.ts`
    - Input: `CitedSource`
    - Displays: type icon (book/gavel/document) + truncated reference
    - Click → navigate to `/normas/{id}`, `/jurisprudencia/{id}`, or `/fuentes-tributarias/{id}`
      depending on `type`

- [ ] **Create** `message-input.component.ts`
    - AppKit 4 textarea (auto-resize, max 4 rows)
    - Send button (disabled while streaming)
    - Enter key sends (Shift+Enter = new line)
    - Char counter (warn at 1000 chars)

- [ ] **Create** `suggested-queries.component.ts`
    - Shown only on new empty conversation
    - 4 example queries (hardcoded in Spanish):
      - "¿Cómo tributa la exportación de servicios en IVA?"
      - "¿Cuál es la alícuota de retención de Ganancias para honorarios?"
      - "Busca jurisprudencia del TFN sobre precios de transferencia"
      - "¿Qué dice la Ley 27.430 sobre dividendos?"
    - Click → sends query automatically
