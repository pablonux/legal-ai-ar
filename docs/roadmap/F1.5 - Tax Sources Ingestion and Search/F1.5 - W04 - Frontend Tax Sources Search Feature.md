# F1.5 - W04 - Frontend Tax Sources Search Feature

> **Feature:** F1.5 - Tax Sources Ingestion and Search
> **Release:** 1.0 | **Sprint:** S01
> **Type:** frontend | **Priority:** High
> **Estimate:** 5 story points
> **Assignable to:** Frontend Dev

---

## Description

Build the Angular 19 `tax-sources` feature: search page with facet filters (organism, document type,
tax branch, date range), a result list, and a document detail view. Reuses components from the
legislation and case-law search features where possible.

---

## Tasks

### Feature scaffold

- [ ] Generate `frontend/projects/features/tax-sources/` using `schema-la:feature`

### Models

- [ ] **Create** `tax-sources.model.ts`
    ```typescript
    export interface SearchTaxSourcesRequest {
      query?: string; organism?: string; documentType?: string;
      taxBranch?: string; publishedFrom?: string; publishedTo?: string;
      page: number; pageSize: number;
    }
    export interface TaxDocumentSummary {
      id: number; organism: string; documentType: string; referenceNumber: string;
      publicationDate?: string; taxBranch: string; title: string; highlightedSummary: string;
    }
    export interface SearchTaxSourcesResponse {
      results: TaxDocumentSummary[]; totalCount: number;
      facets: Record<string, FacetValue[]>; nextCursor?: string;
    }
    export interface TaxDocumentDetail { /* full fields from Contracts */ }
    ```

### Service

- [ ] **Create** `tax-sources.service.ts`
    - `readonly filters = signal<SearchTaxSourcesRequest>(defaults)`
    - `readonly results = signal<SearchTaxSourcesResponse | null>(null)`
    - `readonly loading = signal(false)`
    - `search(overrides?)`, `loadMore()`, `clearFilters()`
    - `getDetail(id: number): Observable<TaxDocumentDetail>`

### Components

- [ ] **Create** `tax-sources-search-shell.component.ts`
    - Two-panel layout: filter sidebar (left) + results (right)
    - Title: "Fuentes Tributarias"

- [ ] **Create** `tax-sources-filters.component.ts`
    - Organism select (ARCA, ARBA, Comisión Arbitral, TFN) — powered by `facets.organism`
    - Document type select — powered by `facets.documentType`
    - Tax branch select — powered by `facets.taxBranch`
    - Date range pickers (publicación desde/hasta)
    - "Limpiar filtros" button

- [ ] **Create** `tax-sources-results.component.ts`
    - Card list of `TaxDocumentSummary`; each card:
      - Organism badge chip (color-coded by organism)
      - Document type tag, reference number
      - Tax branch chip
      - Publication date
      - Highlighted summary (render `<mark>` tags from highlight)
    - "Cargar más" button (cursor pagination) or paged navigation
    - Empty state: "No se encontraron documentos para esta búsqueda"
    - Total count display: "Se encontraron N documentos"

- [ ] **Create** `tax-source-detail.component.ts`
    - Metadata panel: organism, type, reference number, tax branch, period, date
    - Full text section with table of contents if long
    - "Ver documento original" button (link to `originalUrl` or stream from `/document`)
    - "Consultar con asistente" button (deep link to `/asistente?context=tax-source:{id}`)
