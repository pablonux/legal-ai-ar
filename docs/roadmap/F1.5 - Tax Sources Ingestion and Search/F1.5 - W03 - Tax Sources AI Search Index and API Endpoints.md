# F1.5 - W03 - Tax Sources AI Search Index and API Endpoints

> **Feature:** F1.5 - Tax Sources Ingestion and Search
> **Release:** 1.0 | **Sprint:** S01
> **Type:** backend | **Priority:** High
> **Estimate:** 5 story points
> **Assignable to:** Backend Dev

---

## Description

Create the `idx-tax-sources` Azure AI Search index, implement the indexer worker extension that
pushes `TaxDocument` records, and expose the search/detail/facets API endpoints using Minimal API
and Contracts DTOs.
