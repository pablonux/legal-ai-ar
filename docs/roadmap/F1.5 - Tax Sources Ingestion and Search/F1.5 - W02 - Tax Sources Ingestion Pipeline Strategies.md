# F1.5 - W02 - Tax Sources Ingestion Pipeline Strategies

> **Feature:** F1.5 - Tax Sources Ingestion and Search
> **Release:** 1.0 | **Sprint:** S01
> **Type:** backend / worker | **Priority:** High
> **Estimate:** 8 story points
> **Assignable to:** Backend Dev (pipeline experience)

---

## Description

Extend the existing 6-stage ingestion pipeline with new **`IDiscoveryStrategy`** and
**`IDocumentParser`** implementations for the four tax sources: TFN, ARCA, Comisión Arbitral, and
ARBA. Introduce the `TaxDocument` entity, EF migration, and worker configuration. No structural
changes to the pipeline workers — only new strategy classes are added.

Priority order: **TFN first** (has a structured JSON API), then ARCA, then Comisión Arbitral, then
ARBA. Each strategy can be merged independently.

- [ ] **Create** `backend/src/shared/LegalAiAr.Infrastructure/Persistence/Configurations/TaxDocumentConfiguration.cs`
    - Table `TaxDocuments`, unique constraint `UQ_TaxDoc_Source`, indexes on `Organism+PublicationDate` and `TaxBranch`

    - `Task UpsertAsync(TaxDocument doc)` — upsert by `(SourceId, ExternalId)`

### Worker — Discovery strategies

- [ ] **Create** `backend/src/workers/LegalAiAr.Worker.Discoverer/Strategies/TfnDiscoveryStrategy.cs`
    - Calls TFN JSON search API with pagination; emits `DiscoveryItem` messages to the discovery queue
    - Handles rate limiting (respect `Retry-After` header; exponential backoff)
    - Key config: `TfnApi:BaseUrl`, `TfnApi:PageSize`

- [ ] **Create** `backend/src/workers/LegalAiAr.Worker.Discoverer/Strategies/ArcaDiscoveryStrategy.cs`
    - Scrapes `biblioteca.arca.gob.ar` search/pagination pages
    - Extracts links to individual dictamen pages; emits `DiscoveryItem`
    - Respects `robots.txt`; polite crawl delay (1–2 s between requests)

- [ ] **Create** `backend/src/workers/LegalAiAr.Worker.Discoverer/Strategies/ComisionArbitralDiscoveryStrategy.cs`
    - Navigates year/type tree at `ca.gob.ar`; discovers resolution PDFs/pages
    - Emits `DiscoveryItem` with type hint `Resolucion`

- [ ] **Create** `backend/src/workers/LegalAiAr.Worker.Discoverer/Strategies/ArbaDiscoveryStrategy.cs`
    - Navigates category tree at `web.arba.gov.ar/biblioteca-legal`
    - Emits `DiscoveryItem` with type hint based on category name

### Worker — Parser strategies

- [ ] **Create** `backend/src/workers/LegalAiAr.Worker.Parser/Parsers/TaxDictamenParser.cs`
    - Implements `IDocumentParser` for `DocumentType == TaxDocument`
    - Extracts: `ReferenceNumber`, `TaxBranch`, `PublicationDate`, `Title`, `FullText`
    - For PDFs: uses `Azure.AI.DocumentIntelligence` (already in infra) for OCR when HTML not available
    - Identifies `TaxDocumentType` from title heuristics + source metadata
