# F1.2 - W02 - Home API Endpoint and UserActivity

> **Feature:** F1.2 - Home Dashboard
> **Release:** 1.0 | **Sprint:** S01
> **Type:** backend | **Priority:** High
> **Estimate:** 3 story points
> **Assignable to:** Backend Dev

---

## Description

Implement the `GET /api/home` endpoint that returns the personalized home payload (recent activity,
latest alerts, projects placeholder). Introduce the `UserActivity` entity and EF migration. Write
activity records from the search and detail endpoints.

---

## Tasks

### Core — UserActivity entity

- [ ] **Create** `backend/src/shared/LegalAiAr.Core/Entities/UserActivity.cs`
    ```csharp
    public sealed class UserActivity
    {
        public long   Id          { get; private set; }
        public int    UserId      { get; private set; }
        public string Kind        { get; private set; } = default!; // search|norm|ruling|chat
        public string Title       { get; private set; } = default!;
        public string? EntityId   { get; private set; }
        public DateTimeOffset AccessedAt { get; private set; }

        public static UserActivity Create(int userId, string kind, string title, string? entityId = null)
            => new() { UserId = userId, Kind = kind, Title = title, EntityId = entityId,
                       AccessedAt = DateTimeOffset.UtcNow };
    }
    ```
