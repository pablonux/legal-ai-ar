## Roadmap Dashboard (IDE internal tool)

To view roadmap progress after each update:

```bash
node scripts/roadmap-dashboard.js
```

Then open `docs/roadmap/roadmap-dashboard.html` in your browser. No server required — the script embeds the parsed data in the HTML.

The generated file is in `.gitignore`.
